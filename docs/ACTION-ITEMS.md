## Conversion and trust

- [ ] Connect the contact form to **EmailJS**, **Formspree**, or your **CRM**; optional fields (country, trademark name, Amazon yes/no).
- [ ] Add a clear **“next steps”** block: consult → clearance → file → office actions → registration / Brand Registry.
- [ ] Add **social proof** (with permission): more testimonials, partner logos, “as featured in” only if accurate.
- [ ] **Sticky CTA on mobile**: thin bar — “Free 15‑min call” + tap-to-call.

## Content and SEO

- [ ] **Per-blog SEO**: unique `<title>`, meta description, Open Graph / Twitter cards, canonical URLs.
- [ ] **Structured data**: `LegalService` / `Organization` JSON-LD (name, phones, `sameAs`, areas served).
- [ ] **Sitemap** + **`robots.txt`** for static hosting.
- [ ] **FAQ schema** where appropriate for rich results.

## UX and polish

- [ ] **Service deep links**: hash URLs like `#services-us-filing` for sharing.
- [ ] **Reading progress** on long blog posts; **table of contents** for long articles.
- [ ] **Print/PDF stylesheet** for key service summaries.
- [ ] **Cookie / privacy notice** if analytics or chat sets cookies.

## Performance and quality

- [ ] **Compress or convert** hero/illustration PNGs to WebP/AVIF + `srcset`.
- [ ] **Self-host fonts** or subset Google Fonts.
- [ ] **Replace Tailwind CDN** with a built CSS bundle for production (smaller, no runtime).

## International
